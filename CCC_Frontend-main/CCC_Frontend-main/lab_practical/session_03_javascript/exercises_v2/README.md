## 🎯 Tại sao cần v2?

Bộ exercises cũ có vấn đề:
- ❌ Nhảy thẳng vào Todo App với state management, CRUD, localStorage
- ❌ Yêu cầu biết DOM, Events trước khi học xong JS cơ bản
- ❌ Không có hướng dẫn từng bước

Bộ v2 được thiết kế lại:
- ✅ Học theo đúng tiến trình: Biến → Điều kiện → Hàm → Array/Object → DOM → Events
- ✅ Mỗi Tier có 4-5 bài tập, từ dễ đến khó
- ✅ Có code mẫu và thử thách cụ thể
- ✅ Checklist tự đánh giá
