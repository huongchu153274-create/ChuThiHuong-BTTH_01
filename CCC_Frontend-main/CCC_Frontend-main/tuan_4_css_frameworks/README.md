## 🎨 HAI LỰA CHỌN

### 🟦 [Bootstrap 5](./bootstrap/README.md)
**Component-Based CSS Framework** - Components có sẵn, thiết kế nhanh.

**Đặc điểm:**
- ✅ Components có sẵn (Buttons, Cards, Navbar, Forms...)
- ✅ Grid System 12 cột mạnh mẽ
- ✅ JavaScript components (Modals, Dropdowns, Carousel)
- ✅ Cộng đồng lớn, nhiều templates

**Phù hợp khi:**
- Cần prototype nhanh
- Dự án admin panel, dashboard
- Team nhỏ, không có designer
- Cần components phức tạp

---

### 🟩 [TailwindCSS](./tailwindcss/README.md)
**Utility-First CSS Framework** - Utility classes, thiết kế linh hoạt.

**Đặc điểm:**
- ✅ Utility-First approach
- ✅ Bundle size nhỏ (PurgeCSS)
- ✅ Customizable cao
- ✅ Performance tốt

**Phù hợp khi:**
- Cần thiết kế độc đáo
- Muốn control hoàn toàn về styling
- Dự án cần performance tốt
- Team có designer

---

## 🤔 CHỌN FRAMEWORK NÀO?

### Nếu bạn là người mới:
**→ Chọn Bootstrap** - Dễ học hơn, có components sẵn

### Nếu bạn muốn linh hoạt:
**→ Chọn TailwindCSS** - Tự build design, không bị ràng buộc

### Nếu bạn cần prototype nhanh:
**→ Chọn Bootstrap** - Copy-paste components, xong ngay

### Nếu bạn cần performance:
**→ Chọn TailwindCSS** - Bundle size nhỏ, PurgeCSS

---

## 📊 SO SÁNH NHANH

| Tiêu chí | Bootstrap | TailwindCSS |
|----------|-----------|-------------|
| **Phong cách** | Component-Based | Utility-First |
| **Bundle size** | Lớn hơn | Nhỏ (PurgeCSS) |
| **Customization** | Khó hơn (SASS) | Dễ (config file) |
| **Learning curve** | Dễ (copy components) | Dễ (nhớ classes) |
| **Design freedom** | Thấp (theo template) | Cao (tự build) |
| **Tốc độ dev** | Rất nhanh (có sẵn) | Nhanh (sau khi quen) |
