## 🎯 TỔNG QUAN VỀ VUE.JS

### Đặc điểm nổi bật:
- **Progressive Framework:** Có thể tích hợp từng phần vào dự án hiện có
- **Template Syntax:** Dễ học, giống HTML
- **Reactivity System:** Tự động cập nhật UI khi data thay đổi
- **Composition API:** Cách mới để tổ chức code (Vue 3)

### So sánh với React:
| Tiêu chí | Vue.js | React |
|----------|--------|-------|
| **Learning Curve** | Dễ hơn | Trung bình |
| **Template** | HTML-like | JSX |
| **State Management** | Built-in (Vuex/Pinia) | External (Redux) |
| **Bundle Size** | Nhỏ hơn | Lớn hơn |
