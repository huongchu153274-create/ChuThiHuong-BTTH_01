## 🎯 TỔNG QUAN VỀ ANGULAR

### Đặc điểm nổi bật:
- **Full-Featured Framework:** Routing, HTTP, Forms built-in
- **TypeScript First:** Sử dụng TypeScript làm ngôn ngữ chính
- **Dependency Injection:** Hệ thống DI mạnh mẽ
- **RxJS:** Reactive programming với Observables
- **CLI:** Angular CLI mạnh mẽ

### So sánh với React & Vue:
| Tiêu chí | Angular | React | Vue.js |
|----------|---------|-------|--------|
| **Learning Curve** | Khó nhất | Trung bình | Dễ nhất |
| **TypeScript** | Required | Optional | Optional |
| **Bundle Size** | Lớn nhất | Trung bình | Nhỏ nhất |
| **Use Case** | Enterprise apps | Flexible | Progressive |
