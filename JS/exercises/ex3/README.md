# Exercise 3 — Student Grade Manager

Open `index.html` with a local server. Files:
- `data.js` — students array
- `calculator.js` — `calcGPA`, `classify`, `getTopStudent`
- `renderer.js` — `renderTable(students, container)`
- `app.js` — glue code

Run with a local server, e.g. `python -m http.server` from the `student-app` folder.
